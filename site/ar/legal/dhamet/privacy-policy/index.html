سياسة الخصوصية لتطبيق ظامت

آخر تحديث: 29 يونيو 2026

توضح هذه السياسة كيفية جمع واستخدام وحفظ وحماية البيانات المرتبطة باستخدام تطبيق ظامت / Dhamet

تنطبق هذه السياسة على النسخة الرسمية المنشورة من ظامت عبر نطاقات OuglSoft الرسمية، وعلى الميزات المرتبطة بها، بما في ذلك الحسابات، اللعب ضد الحاسوب، اللعب عبر الإنترنت، المشاهدة، الدردشة، الصوت، لوحة الترتيب، ونماذج التواصل.

1. الجهة المسؤولة عن معالجة البيانات

الجهة المسؤولة عن تشغيل التطبيق ومعالجة البيانات المرتبطة به هي:

العُقل للبرمجيات / El Ougl Software SARL

الموقع الرسمي: www.ouglsoft.com

البريد الإلكتروني: contact@ouglsoft.com

البريد الإلكتروني البديل: info@ouglsoft.com

2. ملخص عملي

يعالج ظامت البيانات اللازمة لتشغيل اللعبة وميزاتها. يشمل ذلك تسجيل الدخول، الملف الشخصي، اللعب عبر الإنترنت، مزامنة المباريات، الدعوات، الدردشة، الصوت، الإحصائيات، الترتيب، التحليلات، وتحسين الأداء.

لا نبيع بيانات المستخدمين. لا نؤجرها. لا نمنحها لأطراف أخرى لأغراضها التجارية أو التسويقية الخاصة. بعض البيانات تمر عبر خدمات تقنية لازمة لتشغيل التطبيق، مثل Firebase وGoogle Analytics وCloudflare وخدمات المتصفح المرتبطة بالصوت.

3. البيانات التي قد يعالجها التطبيق

3.1 بيانات الحساب وتسجيل الدخول

عند إنشاء حساب أو تسجيل الدخول، قد يعالج التطبيق البيانات اللازمة لإدارة الحساب والمصادقة، مثل:

  • البريد الإلكتروني.
  • معرّف المستخدم التقني.
  • طريقة تسجيل الدخول.
  • بيانات تسجيل الدخول عبر Google عند استخدام هذا الخيار.
  • الاسم المستعار.
  • الصورة أو الرمز المختار للملف الشخصي.
  • تاريخ إنشاء الحساب أو آخر نشاط، عند توفره.
  • بيانات الجلسة اللازمة لإبقاء المستخدم متصلًا أو استعادة حالة الدخول.

كلمات المرور لا تُحفظ داخل ملفات التطبيق. تتم المصادقة عبر Firebase Authentication أو الخدمة التقنية المستخدمة لإدارة تسجيل الدخول.

3.2 بيانات الملف الشخصي والإحصائيات

قد يعالج التطبيق بيانات مرتبطة بملف اللاعب، مثل:

  • الاسم المستعار.
  • الرمز أو الصورة المختارة.
  • عدد الانتصارات.
  • عدد الخسائر.
  • عدد التعادلات.
  • النقاط.
  • الترتيب العام.
  • مؤشرات النشاط المرتبطة بالتحديثات والإحصائيات.

تُستخدم هذه البيانات لعرض حساب اللاعب ولوحة الترتيب وإحصائيات اللعب.

3.3 بيانات اللعب عبر الإنترنت

عند استخدام اللعب عبر الإنترنت، يعالج التطبيق بيانات تشغيلية لازمة لإنشاء المباراة ومزامنتها، مثل:

  • حالة تواجد اللاعب.
  • الدعوات المرسلة والمستلمة.
  • معرّف الغرفة أو المباراة.
  • أسماء اللاعبين أو أسماؤهم المستعارة.
  • حالة الرقعة.
  • الدور الحالي.
  • النقلات.
  • حالة المباراة.
  • بيانات المزامنة.
  • بيانات الخروج أو الانقطاع أو إعادة الاتصال.
  • بيانات المشاهدة عند دخول المستخدم بصفة مشاهد.

هذه البيانات ضرورية لتشغيل اللعب عبر الإنترنت ولا تُستخدم لإنشاء ملف تسويقي عن المستخدم.

3.4 بيانات الدردشة النصية

إذا استخدم المستخدم الدردشة النصية داخل المباراة، فقد يعالج التطبيق:

  • نص الرسالة.
  • معرّف المرسل.
  • الاسم المستعار للمرسل.
  • دور المرسل داخل الغرفة، مثل لاعب أو مشاهد.
  • وقت الإرسال.
  • مؤشرات القراءة أو آخر قراءة عند توفرها.

لا ينبغي استخدام الدردشة لإرسال بيانات حساسة أو خاصة. قد تبقى الرسائل مدة مرتبطة بحياة الغرفة أو عمليات التنظيف الفني.

3.5 بيانات الصوت والميكروفون

إذا فعّل المستخدم المحادثة الصوتية، يطلب المتصفح إذن استخدام الميكروفون. لا يستطيع التطبيق استخدام الميكروفون دون إذن من المتصفح.

تُستخدم المحادثة الصوتية في الزمن الحقيقي عبر تقنيات WebRTC. قد يعالج التطبيق بيانات تقنية لازمة لتأسيس الاتصال واستمراره، مثل:

  • حالة المشاركة في الصوت.
  • حالة الكتم.
  • إشارات الاتصال التقنية.
  • بيانات المشاركين في غرفة الصوت.
  • بيانات الشبكة اللازمة للاتصال المباشر أو غير المباشر بين المستخدمين.

لا يحتفظ التطبيق بتسجيلات صوتية محفوظة داخل الخدمة. لا يُقصد من ميزة الصوت تسجيل المحادثات أو أرشفتها.

يمكن للمستخدم إيقاف الميكروفون من داخل التطبيق أو سحب إذن الميكروفون من إعدادات المتصفح.

3.6 بيانات المشاهدة

عند دخول المستخدم كمشاهد لمباراة، قد يعالج التطبيق بيانات لازمة للمشاهدة، مثل:

  • معرّف المشاهد.
  • حالة وجوده داخل الغرفة.
  • وقت الدخول أو آخر نشاط.
  • بيانات القراءة أو الدردشة إذا شارك في الدردشة.

3.7 البيانات المحلية داخل المتصفح

قد يخزن التطبيق بيانات على جهاز المستخدم داخل المتصفح، مثل:

  • اللغة المختارة.
  • المظهر أو الوضع البصري.
  • إعدادات اللعب.
  • بيانات الجلسة.
  • بيانات حفظ أو استئناف مباراة.
  • آخر غرفة أو حالة نشطة.
  • تفضيلات الواجهة.

تبقى هذه البيانات على جهاز المستخدم إلى أن يحذفها، أو يمسحها المتصفح، أو تُستبدل ببيانات أحدث.

3.8 بيانات التحليلات

قد يستخدم التطبيق Google Analytics أو أدوات قياس مشابهة لفهم طريقة استخدام التطبيق وتحسين الأداء والاستقرار. قد تشمل بيانات التحليلات:

  • بيانات الجهاز والمتصفح.
  • الصفحات أو الشاشات المستخدمة.
  • أحداث الاستخدام العامة.
  • بيانات تقريبية عن الأداء.
  • بيانات مرتبطة بالجلسة.
  • عناوين IP أو بيانات اتصال يعالجها مزود الخدمة وفق سياساته.

لا تُستخدم التحليلات لبيع بيانات المستخدمين أو إنشاء استهداف إعلاني داخل التطبيق.

3.9 بيانات التواصل

عند إرسال رسالة عبر صفحة التواصل، قد يعالج التطبيق أو موقع الشركة:

  • الاسم.
  • البريد الإلكتروني.
  • موضوع الرسالة.
  • نص الرسالة.
  • بيانات تقنية لازمة لإرسال النموذج وحمايته من إساءة الاستخدام.

تُستخدم هذه البيانات للرد على الطلب أو الملاحظة أو البلاغ، ولا تُستخدم لإنشاء حساب داخل ظامت أو نشر الرسالة داخل التطبيق.

4. أسباب معالجة البيانات

تُعالج البيانات للأغراض التالية:

  • إنشاء الحساب وتسجيل الدخول.
  • تشغيل الملف الشخصي.
  • عرض الإحصائيات ولوحة الترتيب.
  • تشغيل اللعب ضد الحاسوب.
  • تشغيل اللعب عبر الإنترنت.
  • إدارة الدعوات والغرف والمباريات.
  • مزامنة حالة اللعب بين اللاعبين.
  • تمكين المشاهدة عند توفرها.
  • تشغيل الدردشة النصية.
  • تشغيل المحادثة الصوتية عند تفعيلها.
  • حفظ الإعدادات واستئناف اللعب.
  • تحسين الأداء والاستقرار.
  • تطوير منطق اللعب ضد الحاسوب.
  • حماية الخدمة من إساءة الاستخدام.
  • استقبال طلبات التواصل والدعم.
  • الامتثال للمتطلبات القانونية أو الأمنية عند الحاجة.

5. الخدمات الخارجية

يعتمد ظامت على خدمات خارجية لازمة للتشغيل، منها:

5.1 Firebase Authentication

تُستخدم لإدارة تسجيل الدخول والحسابات، بما في ذلك تسجيل الدخول بالبريد الإلكتروني أو عبر Google عند توفره.

5.2 Firebase Realtime Database

تُستخدم لمزامنة البيانات اللازمة للتطبيق، مثل الحسابات، الملفات الشخصية، اللاعبين المتصلين، الدعوات، الغرف، المباريات، الدردشة، المشاهدة، الصوت، الإحصائيات، ولوحة الترتيب.

5.3 Google Analytics

تُستخدم لقياس الاستخدام وتحسين الأداء وتجربة المستخدم.

5.4 Cloudflare

تُستخدم لاستضافة الموقع أو تقديمه عبر الشبكة، وقد تُستخدم وظائف Cloudflare لمعالجة نماذج التواصل أو بعض الطلبات التقنية.

5.5 WebRTC وخدمات الشبكة المرتبطة به

تُستخدم لتشغيل الصوت في الزمن الحقيقي. قد تتضمن هذه العملية تبادل بيانات تقنية بين المتصفحات أو عبر خوادم مساعدة بحسب حالة الاتصال.

6. مشاركة البيانات

لا نشارك بيانات المستخدمين ولا نبيع بيانات المستخدمين ولا نؤجرها. لكن إذا كانت البيانات تمر عبر مزودي الخدمات التقنية اللازمين لتشغيل التطبيق، مثل خدمات الاستضافة، المصادقة، قواعد البيانات، التحليلات، البريد، أو الصوت، فتلك الجهة الخارجية تتحكم بالبيانات التي تمر بها وتستخدم سياساتها الخاصة في التعامل مع البيانات.

7. مدة الاحتفاظ بالبيانات

تختلف مدة الاحتفاظ بحسب نوع البيانات والغرض منها:

  • بيانات الحساب والملف الشخصي تبقى ما دام الحساب قائمًا أو ما دامت لازمة لتشغيله.
  • بيانات الإحصائيات والترتيب تبقى ما دام الحساب أو السجل المرتبط بها قائمًا.
  • بيانات الغرف والمباريات والدردشة والصوت والمشاهدة تشغيلية بطبيعتها وتُزال عند انتهاء المباراة أو مغادرة الغرفة أو ضمن عمليات تنظيف دورية.
  • بيانات التواصل تبقى مدة لازمة للرد والمتابعة وحفظ سجل الدعم عند الحاجة.
  • بيانات التحليلات تخضع لإعدادات الخدمة المستخدمة وسياسات مزودها.
  • البيانات المحلية في المتصفح تبقى على جهاز المستخدم حتى يحذفها أو يمسحها المتصفح.

8. حذف الحساب والبيانات

يمكن للمستخدم حذف حسابه من الأدوات المتاحة داخل التطبيق عند توفرها. عند حذف الحساب، يعمل التطبيق على حذف البيانات المباشرة المرتبطة بالحساب من المسارات الأساسية التي يديرها، مثل الملف الشخصي، الترتيب، حالة اللاعب، والدعوات.

قد لا يؤدي حذف الحساب إلى حذف كل أثر تقني أو تاريخي فورًا، مثل:

  • بيانات موجودة في نسخ احتياطية أو سجلات مؤقتة.
  • بيانات مجمعة أو مشتقة لا تحدد المستخدم مباشرة.
  • رسائل أو آثار مرتبطة بغرف أو مباريات انتهت قبل الحذف.
  • بيانات يحتفظ بها مزود خدمة خارجي وفق سياساته.
  • بيانات يلزم الاحتفاظ بها لحماية الخدمة أو الامتثال القانوني.

9. خيارات المستخدم

يمكن للمستخدم:

  • استخدام اسم مستعار لا يكشف هويته الحقيقية.
  • تعديل بيانات الملف الشخصي المتاحة.
  • حذف الحساب من داخل التطبيق بجميع بياناته.
  • حذف بيانات المتصفح المحلية من إعدادات المتصفح.
  • سحب إذن الميكروفون من إعدادات المتصفح.
  • عدم استخدام الدردشة أو الصوت.
  • عدم تسجيل الدخول عبر Google والدخول كضيف.
  • التواصل معنا لطلبات الخصوصية أو الدعم.

10. الدردشة والصوت والمحتوى الذي يقدمه المستخدم

يتحمل المستخدم مسؤولية ما يكتبه أو يرسله أو يقوله داخل الدردشة أو الصوت. يجب عدم مشاركة بيانات حساسة، كلمات مرور، معلومات مالية، بيانات شخصية للغير، أو محتوى مخالف للقانون أو الآداب أو حقوق الآخرين.

قد نراجع أو نقيّد أو نحذف محتوى أو نعلّق وصول مستخدم إذا كان ذلك لازمًا لحماية التطبيق أو المستخدمين أو حقوق الشركة.

11. الأطفال والقُصّر

ظامت لعبة عامة عبر الويب وليست موجّهة خصيصًا للأطفال. إذا كان المستخدم قاصرًا وفق القوانين المطبقة في بلده، فيجب أن يستخدم التطبيق بموافقة ولي أمره أو تحت إشرافه.

لا نطلب عمدًا من الأطفال تقديم بيانات حساسة. إذا اعتقد ولي الأمر أن طفلًا قدّم بيانات شخصية دون موافقة مناسبة، يمكنه التواصل معنا لطلب المراجعة أو الحذف.

12. الأمان

نستخدم تدابير فنية وتنظيمية معقولة لحماية البيانات، مثل الاتصال الآمن، ضوابط الوصول، خدمات مصادقة، وقواعد أمان قاعدة البيانات. لا يمكن لأي خدمة تعمل عبر الإنترنت ضمان أمان كامل أو مطلق.

يتحمل المستخدم مسؤولية حماية بيانات دخوله، واستخدام كلمة مرور قوية، وعدم مشاركة حسابه مع الآخرين.

13. الروابط الخارجية

قد يحتوي التطبيق أو الموقع على روابط إلى مواقع أو خدمات خارجية، مثل موقع الشركة المطورة OuglSoft أو خدمات Google أو غيرها. تخضع هذه المواقع والخدمات لسياسات الخصوصية والشروط الخاصة بها.

14. النقل والمعالجة الدولية

قد تُعالج البيانات عبر خدمات سحابية تعمل في دول مختلفة عن بلد المستخدم. باستخدام التطبيق، يقر المستخدم بأن البيانات قد تُعالج عبر بنية تقنية دولية بحسب الخدمات المستخدمة لتشغيل التطبيق.

15. التغييرات على هذه السياسة

قد نحدّث هذه السياسة عند تغيير ميزات التطبيق أو طريقة معالجة البيانات أو الخدمات الخارجية المستخدمة أو المتطلبات القانونية. تُنشر النسخة المحدثة مع تاريخ آخر تحديث. استمرار استخدام التطبيق بعد نشر التحديث يعني الاطلاع على النسخة الجديدة.
