شروط استخدام تطبيق ظامت

آخر تحديث: 29 يونيو 2026

تنظم هذه الشروط استخدام تطبيق ظامت / Dhamet.

باستخدام التطبيق، يوافق المستخدم على هذه الشروط، وعلى سياسة الخصوصية، وعلى قواعد اللعبة المنشورة ضمن التطبيق. إذا لم يوافق المستخدم على هذه الشروط، يجب عليه عدم استخدام التطبيق.

1. التعريفات

في هذه الشروط:

  • التطبيق يعني تطبيق ظامت / Dhamet المنشور عبر الويب.
  • الشركة تعني العُقل للبرمجيات / El Ougl Software SARL.
  • المستخدم يعني كل شخص يدخل إلى التطبيق أو يستخدمه.
  • الحساب يعني حساب المستخدم داخل التطبيق أو عبر خدمة المصادقة المستخدمة.
  • المحتوى يعني النصوص، الرسائل، الأسماء، الصور، الرموز، النقلات، بيانات اللعب، أو أي بيانات يرسلها أو ينشئها المستخدم داخل التطبيق.
  • القواعد تعني قواعد لعبة ظامت المنشورة داخل التطبيق.

2. طبيعة الخدمة

يوفر ظامت نسخة رقمية من لعبة ظامت الموريتانية عبر المتصفح. قد يشمل التطبيق ميزات مثل:

  • اللعب ضد الحاسوب.
  • اللعب عبر الإنترنت.
  • إرسال واستقبال الدعوات.
  • مشاهدة بعض المباريات عند توفرها.
  • حساب لاعب وملف شخصي.
  • إحصائيات وترتيب.
  • دردشة نصية داخل المباراة.
  • محادثة صوتية عند تفعيلها.
  • حفظ أو استئناف بعض الحالات محليًا.
  • دعم العربية والإنجليزية والفرنسية.

قد تتغير الميزات أو تتوقف أو تُقيّد أو تُحدّث في أي وقت بحسب احتياجات التشغيل والتطوير والأمان.

3. الحساب وتسجيل الدخول

قد يتطلب استخدام بعض ميزات ظامت إنشاء حساب أو تسجيل الدخول. يلتزم المستخدم بتقديم بيانات صحيحة بالقدر اللازم لتشغيل الحساب، وبحماية بيانات الدخول الخاصة به.

لا يجوز للمستخدم:

  • استخدام حساب شخص آخر دون إذن.
  • مشاركة الحساب بطريقة تسبب إساءة استخدام أو ضررًا للتطبيق.
  • انتحال صفة شخص أو جهة.
  • استخدام اسم مستعار مضلل أو مسيء أو مخالف.
  • إنشاء حسابات متعددة للتحايل على القيود أو الترتيب أو إجراءات الأمان.

يجوز للشركة تقييد الحساب أو تعليقه أو حذفه عند مخالفة هذه الشروط أو إساءة استخدام التطبيق.

4. الترخيص الممنوح للمستخدم

تمنح الشركة المستخدم ترخيصًا محدودًا، شخصيًا، غير حصري، غير قابل للتحويل، وقابلًا للإلغاء، للوصول إلى النسخة الرسمية المنشورة من ظامت واستخدامها للأغراض الشخصية وغير التجارية وفق هذه الشروط.

لا يمنح هذا الترخيص أي حق في ملكية التطبيق أو شفرته أو تصميمه أو أصوله أو نماذجه أو محتواه أو علاماته أو وثائقه.

5. الملكية الفكرية

جميع حقوق التطبيق محفوظة للشركة، ويشمل ذلك:

  • الشفرة البرمجية.
  • التصميم والواجهة.
  • الأصول الرقمية.
  • النصوص والوثائق.
  • نماذج أو منطق اللعب ضد الحاسوب.
  • البنية التقنية.
  • العلامات والأسماء والشعارات المرتبطة بالتطبيق.
  • أي تحسينات أو تحديثات أو أعمال مشتقة تطورها الشركة.

إتاحة التطبيق مجانًا عبر الويب لا تعني أنه مفتوح المصدر، ولا تمنح المستخدم حق نسخه أو تعديله أو إعادة نشره أو استضافته أو تشغيل نسخة أخرى منه.

6. الاستخدام المسموح

يجوز للمستخدم استخدام التطبيق للعب والتعلم والتدريب والمشاهدة والتواصل داخل الميزات المتاحة، بشرط الالتزام بالقواعد، واحترام المستخدمين الآخرين، وعدم الإضرار بالخدمة أو محاولة تجاوز قيودها.

7. الاستخدامات المحظورة

لا يجوز للمستخدم، بشكل مباشر أو غير مباشر:

  • نسخ التطبيق أو استخراج شفرته أو أصوله أو نماذجه.
  • إجراء هندسة عكسية أو تفكيك أو تحليل غير مصرح به للتطبيق.
  • تعديل التطبيق أو إنشاء نسخة مشتقة أو إعادة نشره.
  • استضافة نسخة أخرى من التطبيق دون إذن كتابي من الشركة.
  • استخدام التطبيق استخدامًا تجاريًا دون إذن.
  • تعطيل الخدمة أو إبطاؤها أو مهاجمتها أو استغلال ثغراتها.
  • العبث بمزامنة المباريات أو الغرف أو الدعوات أو الترتيب.
  • استخدام أدوات آلية أو بوتات أو سكربتات للعب أو جمع البيانات أو التلاعب.
  • الغش أو استغلال أخطاء تقنية للحصول على نتيجة غير عادلة.
  • انتحال شخصية مستخدم آخر.
  • إرسال رسائل مزعجة أو مسيئة أو تهديدية أو احتيالية.
  • استخدام الدردشة أو الصوت للتحرش أو الإساءة أو نشر معلومات خاصة بالآخرين.
  • نشر محتوى غير قانوني أو ضار أو ينتهك حقوق الغير.
  • محاولة الوصول إلى بيانات أو حسابات أو غرف لا يملك المستخدم حق الوصول إليها.
  • تجاوز إجراءات الحماية أو القيود الفنية.
  • استخدام التطبيق بطريقة تخالف القانون أو هذه الشروط.

8. قواعد اللعبة

تخضع المباريات لقواعد ظامت المنشورة داخل التطبيق. يجب على المستخدم قراءة القواعد وفهمها قبل الاعتراض على نتيجة أو نقلة أو عقوبة داخل اللعبة.

تتضمن اللعبة قواعد خاصة مثل الافتتاح الإجباري، الأسر، السوفلة، حركة القطع، وحالات الفوز أو التعادل. قد يطبق التطبيق هذه القواعد آليًا أو يوفر أدوات لتطبيق بعضها داخل المباراة.

عند وجود اختلاف بين فهم المستخدم للعبة وبين القواعد المنشورة في التطبيق، تكون القواعد المنشورة داخل التطبيق هي المرجع التشغيلي للنسخة الرقمية، إلى أن يتم تحديثها أو تصحيحها.

9. اللعب عبر الإنترنت

يعتمد اللعب عبر الإنترنت على اتصال المستخدم، والمتصفح، وخدمات المزامنة، وحالة الطرف الآخر. قد تحدث حالات تأخير أو انقطاع أو اختلاف مؤقت في الحالة أو فشل في الدعوات أو الغرف.

لا تضمن الشركة أن كل مباراة عبر الإنترنت ستكتمل دون انقطاع أو خطأ. يجوز للتطبيق إنهاء غرفة أو حذف بيانات تشغيلية أو إعادة ضبط حالة معينة عند الحاجة الفنية أو الأمنية.

10. الدردشة النصية

الدردشة داخل المباراة مخصصة للتواصل المرتبط باللعب. يتحمل المستخدم مسؤولية الرسائل التي يرسلها.

يحظر استخدام الدردشة في:

  • الإساءة أو السب أو التهديد.
  • التحرش أو الاستهداف الشخصي.
  • نشر بيانات خاصة عن الغير.
  • إرسال روابط خادعة أو ضارة.
  • الاحتيال أو طلب كلمات المرور أو البيانات الحساسة.
  • نشر محتوى غير قانوني أو مخالف للآداب أو حقوق الآخرين.
  • إغراق الغرفة برسائل متكررة أو مزعجة.

يجوز للشركة حذف محتوى أو تقييد حساب أو تعطيل ميزة عند إساءة استخدام الدردشة.

11. المحادثة الصوتية

المحادثة الصوتية ميزة اختيارية داخل بعض أوضاع اللعب. يتطلب استخدامها إذن الميكروفون من المتصفح. يمكن للمستخدم كتم الميكروفون أو تعطيل الإذن من إعدادات المتصفح.

يحظر استخدام الصوت في:

  • الإساءة أو التهديد أو التحرش.
  • تسجيل أو نشر صوت الآخرين دون إذن إذا كان ذلك مخالفًا للقانون.
  • مشاركة بيانات حساسة أو كلمات مرور.
  • استخدام الصوت للتشويش أو المضايقة أو تعطيل تجربة اللعب.

لا تضمن الشركة جودة الاتصال الصوتي أو استمراره، لأنه يعتمد على المتصفح، الجهاز، الشبكة، والخدمات التقنية الوسيطة.

12. المحتوى الذي يقدمه المستخدم

يحتفظ المستخدم بما يملكه من حقوق في المحتوى الذي يقدمه، لكنه يمنح الشركة ترخيصًا محدودًا لاستخدام هذا المحتوى بالقدر اللازم لتشغيل التطبيق، عرضه، نقله، تخزينه، مزامنته، معالجته، حذفه، أو حماية الخدمة.

يشمل ذلك، حسب الحالة، الأسماء المستعارة، الرسائل، بيانات اللعب، النقلات، الإحصائيات، وبيانات الغرف.

لا يجوز للمستخدم إرسال محتوى لا يملك حق استخدامه أو يخالف حقوق الآخرين.

13. الإحصائيات ولوحة الترتيب

قد يعرض التطبيق إحصائيات وترتيبًا للاعبين. هذه البيانات تشغيلية وليست سجلًا رسميًا أو مضمونًا.

قد تتأثر الإحصائيات والترتيب بأخطاء تقنية، انقطاع الاتصال، تحديثات النظام، حذف الحساب، عمليات صيانة، أو محاولات إساءة استخدام. يجوز للشركة تصحيح أو إعادة حساب أو حذف أو تجاهل بيانات ترتيب عند الحاجة.

لا يملك المستخدم حقًا مكتسبًا في ترتيب أو نقطة أو إحصائية معينة.

15. التواصل والدعم

يمكن للمستخدم التواصل مع الشركة عبر صفحة التواصل الرسمية أو البريد الإلكتروني. لا تلتزم الشركة بالرد على كل رسالة، لكنها قد تستخدم الرسائل لتحسين الخدمة أو معالجة البلاغات أو الدعم.

يجب ألا تتضمن رسائل التواصل بيانات حساسة غير لازمة، مثل كلمات المرور أو بيانات مالية أو معلومات خاصة بالغير.

16. تعليق الوصول وإنهاء الاستخدام

يجوز للشركة، دون إشعار مسبق عند الحاجة، أن تتخذ أيًا من الإجراءات التالية:

  • تقييد الوصول إلى ميزة معينة.
  • تعليق الحساب.
  • حذف الحساب أو بعض البيانات المرتبطة به.
  • تعطيل الدردشة أو الصوت.
  • إزالة محتوى مخالف.
  • منع استخدام التطبيق مؤقتًا أو دائمًا.
  • اتخاذ إجراءات فنية لحماية الخدمة.

يجوز اتخاذ هذه الإجراءات عند مخالفة الشروط، إساءة الاستخدام، محاولة الغش، تهديد الأمان، انتهاك حقوق الآخرين، أو الإضرار بتجربة المستخدمين.

17. التحديثات والتغييرات

يجوز للشركة تحديث التطبيق أو تغيير ميزاته أو تعديل الواجهة أو القواعد التشغيلية أو الشروط أو سياسة الخصوصية. قد تؤدي التحديثات إلى تغيير طريقة اللعب، حفظ البيانات، المزامنة، الترتيب، أو الحسابات.

استمرار استخدام التطبيق بعد نشر أي تحديث يعني قبول النسخة المحدثة من الشروط.

18. التوفر والانقطاع

يُقدم التطبيق بحسب الإمكانات الفنية المتاحة. لا تضمن الشركة أن التطبيق سيكون متاحًا دائمًا أو خاليًا من الأخطاء أو مناسبًا لكل جهاز أو متصفح أو اتصال.

قد يتوقف التطبيق كليًا أو جزئيًا بسبب الصيانة، التحديث، أخطاء الشبكة، خدمات خارجية، قيود المتصفح، أو ظروف خارجة عن سيطرة الشركة.

19. الخدمات الخارجية

يعتمد التطبيق على خدمات خارجية مثل خدمات المصادقة، قواعد البيانات، التحليلات، الاستضافة، البريد، أو الاتصال الصوتي. تخضع هذه الخدمات لشروطها وسياساتها الخاصة.

لا تتحمل الشركة مسؤولية انقطاع أو تغيير أو خطأ ناتج عن خدمة خارجية، إلا في الحدود التي يفرضها القانون.

20. إخلاء المسؤولية

يُقدّم التطبيق كما هو وكما يتوفر. لا تقدم الشركة ضمانات صريحة أو ضمنية بشأن خلو التطبيق من الأخطاء، أو استمراره، أو ملاءمته لغرض معين، أو دقة كل نتيجة أو ترتيب أو إحصائية.

إلى الحد الذي يسمح به القانون، لا تتحمل الشركة المسؤولية عن أي أضرار مباشرة أو غير مباشرة أو عرضية أو تبعية تنتج عن استخدام التطبيق أو عدم القدرة على استخدامه أو فقدان بيانات أو انقطاع اتصال أو أخطاء مزامنة أو تصرفات مستخدمين آخرين.

21. حدود المسؤولية

إذا ثبتت مسؤولية الشركة في حالة معينة، فإن مسؤوليتها تكون محدودة بالقدر الذي يسمح به القانون، وبما يتناسب مع طبيعة الخدمة المجانية، ودون المساس بأي حقوق لا يجوز استبعادها قانونًا.

22. القُصّر

إذا كان المستخدم قاصرًا وفق القانون المطبق عليه، فيجب أن يستخدم التطبيق بموافقة ولي أمره أو تحت إشرافه. يتحمل ولي الأمر مسؤولية متابعة استخدام القاصر للتطبيق وميزات الدردشة أو الصوت.

23. القانون والاختصاص

تخضع هذه الشروط للقوانين المطبقة على الشركة ومكان تسجيلها، ما لم تنص قواعد قانونية آمرة على خلاف ذلك. في حال النزاع، يتم السعي أولًا إلى حله وديًا عبر التواصل الرسمي قبل اللجوء إلى أي إجراءات أخرى.

24. بطلان أحد البنود

إذا اعتُبر أي بند من هذه الشروط غير صالح أو غير قابل للتنفيذ، يبقى باقي الشروط نافذًا بالقدر الممكن.
