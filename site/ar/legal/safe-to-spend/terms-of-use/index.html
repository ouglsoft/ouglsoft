شروط استخدام تطبيق Safe-to-Spend

تحكم هذه الشروط استخدام تطبيق Safe-to-Spend. باستخدام التطبيق أو تثبيته أو الوصول إلى أي من وظائفه، يوافق المستخدم على الالتزام بهذه الشروط. إذا لم يوافق المستخدم على هذه الشروط، فيجب عليه عدم استخدام التطبيق.

تاريخ النفاذ: 4 يوليو 2026

1. غرض التطبيق

Safe-to-Spend تطبيق لإدارة الإنفاق الشخصي يتيح تسجيل الحسابات، المعاملات، التصنيفات، الميزانيات، التصنيفات المستبعدة، النسخ الاحتياطية، والعناصر المحذوفة، ومراجعة هدف الشهر، هامش الأمان، معدل الإنفاق، التوقع الذكي، المقارنة مع الشهر السابق، المقارنة مع آخر 12 شهرًا متاحة، أثر الميزانية، والتحليل الذي يختاره المستخدم.

2. الاستخدام الشخصي

يوفر التطبيق للاستخدام الشخصي غير التجاري. لا يُعد خدمة مصرفية، أو خدمة دفع، أو منتجًا استثماريًا، أو نظامًا محاسبيًا، أو خدمة ضريبية، أو خدمة قانونية، أو منصة لإدارة الأعمال.

لا يجوز استخدام التطبيق لتقديم خدمات مالية أو محاسبية أو إدارية أو استشارية أو خدمات إدارة بيانات لأي طرف ثالث دون إذن كتابي مسبق من OuglSoft.

3. ترخيص الاستخدام

تمنح OuglSoft المستخدم ترخيصًا محدودًا، قابلًا للإلغاء، غير حصري، غير قابل للنقل، وغير قابل للترخيص من الباطن، لاستخدام التطبيق لأغراض شخصية وفقًا لهذه الشروط والقانون المعمول به.

لا تنتقل إلى المستخدم أي ملكية أو حقوق ملكية فكرية أو حقوق في الشيفرة المصدرية أو حقوق تجارية أو حقوق توزيع بسبب تثبيت التطبيق أو استخدامه.

4. الملكية الفكرية

التطبيق، بما في ذلك اسمه وتصميمه وواجهاته ونصوصه ورسوماته وأيقوناته وبنيته وحساباته وشيفرته المصدرية والمترجمة ووثائقه وأي مواد مرتبطة به، مملوك لـ OuglSoft أو للجهات المرخصة لها، ومحمي بموجب قوانين الملكية الفكرية المعمول بها.

5. الأفعال المحظورة

لا يجوز للمستخدم نسخ التطبيق أو نشره أو بيعه أو تأجيره أو ترخيصه من الباطن أو إعادة توزيعه أو تعديله أو ترجمته أو تفكيكه أو عكس هندسته أو استخراج شيفرته أو إزالة إشعارات الحقوق منه أو إنشاء أعمال مشتقة منه، إلا في الحدود التي يسمح بها القانون صراحة.

لا يجوز استخدام التطبيق بطريقة غير مشروعة، أو تعطيل تشغيله، أو تجاوز آليات الحماية فيه، أو انتحال صفة الغير، أو استخدامه بطريقة تنتهك حقوق OuglSoft أو حقوق أي طرف ثالث.

6. بيانات المستخدم ودقتها

يتحمل المستخدم مسؤولية دقة البيانات التي يدخلها في التطبيق واكتمالها وتصنيفها ونسخها احتياطيًا وتصديرها واستيرادها وحذفها وتفسير النتائج الناتجة عنها.

تعتمد التقارير والأهداف والتوقعات والمقارنات وهوامش الأمان والسلاسل الشهرية وأثر الميزانية ونتائج التحليل على البيانات التي يدخلها المستخدم، وقد تكون غير دقيقة إذا كانت البيانات ناقصة أو خاطئة أو مكررة أو مستبعدة أو محذوفة أو مستوردة بطريقة غير صحيحة.

7. عدم تقديم مشورة مالية أو ضريبية أو قانونية

يوفر التطبيق أدوات تنظيم وحساب ومقارنة وعرض بصري فقط. لا يقدم التطبيق مشورة مالية أو استثمارية أو ضريبية أو محاسبية أو قانونية، ولا يقدم توصية بإجراء أي معاملة أو الامتناع عنها.

يبقى المستخدم وحده مسؤولًا عن قراراته المالية، وعليه مراجعة أي معلومات بشكل مستقل قبل الاعتماد عليها.

8. التوقعات والمقارنات

التوقع الذكي، معدل الإنفاق، أهداف الشهر، هامش الأمان، المقارنة مع الشهر السابق، المقارنة مع آخر 12 شهرًا متاحة، أثر الميزانية، وأثر التصنيفات هي تقديرات أو حسابات مبنية على البيانات المتاحة داخل التطبيق.

هذه النتائج أدوات معلوماتية فقط، ولا تمثل ضمانًا للإنفاق المستقبلي أو الادخار أو النتائج المالية أو الالتزام بالميزانية.

9. التصنيفات المستبعدة

التصنيفات المستبعدة ميزة يتحكم فيها المستخدم، وتهدف إلى إخراج المصروفات الاستثنائية أو غير القابلة للمقارنة من بعض حسابات المقارنة الشهرية. يتحمل المستخدم مسؤولية اختيار الاستبعادات المناسبة ومراجعة أثرها على التقارير والنتائج.

10. البيانات الافتراضية

قد يتضمن التطبيق حسابات وتصنيفات ومعاملات وميزانيات وسجلات شهرية افتراضية لأغراض الشرح. يمكن للمستخدم حذف البيانات الافتراضية من داخل التطبيق عند توفر هذه الوظيفة. لا تمثل البيانات الافتراضية مشورة مالية ولا تُعد سجلًا ماليًا حقيقيًا للمستخدم.

11. النسخ الاحتياطي والتصدير والاستيراد والحذف

يتحمل المستخدم مسؤولية إنشاء النسخ الاحتياطية وتخزينها وحمايتها واستعادتها وحذفها، ومسؤولية الملفات التي يصدرها أو يستوردها. قد يؤدي الاستيراد إلى تعديل السجلات أو تكرارها أو تغييرها حسب الملف وعملية الاستيراد.

يجب على المستخدم مراجعة البيانات المستوردة أو المستعادة قبل الاعتماد عليها. لا تتحمل OuglSoft مسؤولية فقدان أو كشف أو تلف أو حذف أو سوء استخدام أي ملف قام المستخدم بتصديره أو استيراده.

12. قفل التطبيق وأمان الجهاز

كلمة المرور أو قفل التطبيق، عند توفرهما، ميزة محلية للتحكم في الدخول. لا يضمن ذلك حماية مطلقة، ولا يغني عن حماية الجهاز، والتشفير، وتحديثات نظام التشغيل، والتعامل المسؤول مع الملفات.

13. التحديثات والتغييرات

يجوز لـ OuglSoft تحديث التطبيق أو تعديله أو تعليق أي جزء منه أو إزالته أو إيقافه، بما في ذلك الحسابات والشاشات والبيانات الافتراضية ووظائف الاستيراد والتصدير والنسخ الاحتياطي وعناصر الواجهة. استمرار استخدام التطبيق بعد التحديث يعني استمرار قبول التطبيق المحدث والشروط السارية.

14. المنصات والخدمات الخارجية

إذا تم تنزيل التطبيق من متجر تطبيقات أو استخدامه مع نظام تشغيل أو ميزة جهاز أو خدمة تخزين أو مدير ملفات أو متصفح أو خدمة مشاركة، فإن استخدام تلك المنصات والخدمات يخضع لشروطها وسياساتها الخاصة.

15. إخلاء الضمانات

إلى أقصى حد يسمح به القانون، يوفر التطبيق كما هو وكما هو متاح، دون أي ضمانات صريحة أو ضمنية أو قانونية، بما في ذلك ضمان الدقة، أو الملاءمة لغرض معين، أو التشغيل المستمر، أو خلوه من الأخطاء، أو التوافق، أو التوفر.

16. تحديد المسؤولية

إلى أقصى حد يسمح به القانون، لا تتحمل OuglSoft مسؤولية أي أضرار غير مباشرة أو عرضية أو خاصة أو تبعية أو عقابية أو نموذجية، أو أي فقدان بيانات أو ملفات، أو إدخالات خاطئة، أو حسابات غير دقيقة، أو قرارات مالية، أو مشكلات في الجهاز، أو وصول غير مصرح به، أو سوء استخدام للملفات المصدّرة، أو فشل في الاستيراد أو النسخ الاحتياطي، أو الاعتماد على مخرجات التطبيق.

17. إنهاء الترخيص

ينتهي ترخيص المستخدم تلقائيًا إذا خالف هذه الشروط. عند الإنهاء، يجب على المستخدم التوقف عن استخدام التطبيق وحذف أي نسخ أو مواد غير مصرح بها.

18. القانون الحاكم

تخضع هذه الشروط للقوانين المعمول بها في مقر OuglSoft، ما لم تفرض قوانين حماية المستهلك الإلزامية في مكان إقامة المستخدم خلاف ذلك.

19. تغييرات هذه الشروط

يجوز لـ OuglSoft تحديث هذه الشروط عند الحاجة. تسري الشروط المحدثة من تاريخ نشرها في الموقع الرسمي المخصص للتطبيق.
