# موقع العُقل للبرمجيات

هذه نسخة منظمة متعددة اللغات من موقع الشركة. الإنجليزية هي اللغة الافتراضية في جذر الموقع، والفرنسية داخل `fr/`، والعربية داخل `ar/`.

## البنية
- `index.html`: الصفحة الرئيسية الإنجليزية.
- `pages/`: صفحات الشركة الداخلية بالإنجليزية.
- `products/`: صفحات المنتجات بالإنجليزية.
- `fr/`: النسخة الفرنسية.
- `ar/`: النسخة العربية.
- `assets/css/`, `assets/js/`, `assets/images/`: ملفات التصميم والبرمجة والصور.
- `legal/`: صفحات قانونية مباشرة لتطبيقات الهاتف، غير مربوطة من قوائم الموقع العامة.

## قبل النشر
لا توجد قيم مؤقتة ظاهرة داخل الصفحات. روابط تحميل تطبيقات الهاتف توجه إلى صفحة التواصل إلى حين اعتماد روابط المتاجر الرسمية.
