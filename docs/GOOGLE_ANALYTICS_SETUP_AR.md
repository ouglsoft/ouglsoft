# Google Analytics

تمت إضافة Google Analytics 4 إلى صفحات HTML في موقع الشركة باستخدام Google tag اليدوي.

Measurement ID:

```text
G-L3S190MBC9
```

تم وضع الكود مباشرة بعد وسم `<head>` في كل صفحة HTML موجودة وقت التعديل.

عند إضافة صفحات HTML جديدة لاحقًا، يجب إضافة نفس الكود داخل `<head>` أو استخدام قالب مشترك إن تم تحويل الموقع مستقبلًا إلى نظام قوالب.

لا تستخدم Google Tag Manager في هذه النسخة حتى لا يتم تكرار التتبع.
